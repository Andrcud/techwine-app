# Mini App — TechWine (VasonGroup)

Каталог Juclas + Vason внутри Telegram. Статическая страница, бесплатный хостинг.

## Что внутри

- `index.html` — само приложение (один файл, без сборки)
- `catalog.json` — данные: 16 категорий Juclas, 38 аппаратов, 259 продуктов Vason, 28 разделов, 3 языка
- **Цен здесь нет.** Они отдаются только ботом после проверки доступа.

## Развёртывание — 4 шага

### 1. Указать имя бота

В `index.html`, строка ~200:

```js
const BOT = "TechWineVasonBot";   // ← ваш @username без @
```

Это критично: без правильного имени кнопки «Узнать цену» и «Скачать PDF» никуда не приведут.

### 2. Залить на GitHub Pages

1. Создать репозиторий на github.com (можно приватный аккаунт, но **сам репозиторий — public**, иначе Pages не отдаст файлы).
2. Загрузить `index.html` и `catalog.json` в корень.
3. Settings → Pages → Source: `Deploy from a branch` → Branch: `main`, папка `/ (root)` → Save.
4. Через 1–2 минуты получите адрес вида:
   `https://ВАШ_ЛОГИН.github.io/ВАШ_РЕПОЗИТОРИЙ/`

Проверьте, что страница открывается в обычном браузере.

### 3. Прописать адрес боту

В файле `.env` рядом с ботом:

```
WEBAPP_URL=https://ВАШ_ЛОГИН.github.io/ВАШ_РЕПОЗИТОРИЙ/
```

Перезапустить бота. В главном меню появится кнопка **«🚀 Открыть каталог»**, а в поле ввода — кнопка меню **«Каталог»**.

### 4. Зарегистрировать Mini App в BotFather (опционально, но желательно)

`@BotFather` → `/mysettings` → выбрать бота → `Menu Button` → вставить тот же URL.
Тогда приложение открывается прямо из шапки чата.

## Как работает доступ к ценам

```
Mini App (публичный)          Бот (по приглашению)
├── каталог, описания    ──▶  t.me/бот?start=p_E3007793
├── поиск, артикулы           └─▶ is_approved()? ──▶ цены
└── БЕЗ ЦЕН                                     └──▶ отказ + запрос доступа
```

Ссылку на Mini App можно свободно пересылать: ценовую сетку из неё не вытащить.

## Обновление каталога

Когда придёт новый прайс Vason:

1. Положить новые `vason_full_catalog_ru.pdf` и `vason_full_catalog_RO.pdf` рядом с ботом.
2. Перегенерировать `vason_prices.py` и `catalog.json` (скрипт генерации — в истории работы; попросите пересобрать).
3. Залить обновлённый `catalog.json` в репозиторий.

Не редактируйте `catalog.json` и `vason_prices.py` руками — они собираются из PDF.

## Ограничение

Приложение статическое: цены, персонализация и авторизация в нём невозможны by design.
Если понадобится показывать цены **внутри** приложения только одобренным — нужен бэкенд
(Cloudflare Workers, бесплатный тариф) с проверкой подписи `initData`.
